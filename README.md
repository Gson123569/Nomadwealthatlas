## Deploy to GitHub Pages (5 Minutes)

### Step 1: Create a GitHub Account
Go to [github.com](https://github.com) and sign up (it's free).

### Step 2: Create a New Repository
1. Click the **+** button (top right)
2. Select **"New repository"**
3. Name it: `nomadwealthatlas`
4. Make it **Public**
5. Click **"Create repository"**

### Step 3: Upload Files
1. In your new repository, click **"uploading an existing file"**
2. Drag and drop ALL files from this folder
3. Click **"Commit changes"**

### Step 4: Enable GitHub Pages
1. Go to **Settings** (top tab)
2. Click **Pages** (left sidebar)
3. Under "Source", select **Deploy from a branch**
4. Select branch: **main**
5. Select folder: **/(root)**
6. Click **Save**

### Step 5: Wait 2 Minutes
Your site will be live at:
`https://yourusername.github.io/nomadwealthatlas`

(Replace "yourusername" with your actual GitHub username)

## Customization

### Change Colors
Edit `css/style.css` — look for `:root` at the top.

### Change Content
Edit any `.html` file — it's just text with HTML tags.

### Add a Page
1. Copy any existing `.html` file
2. Rename it
3. Edit the content
4. Add a link in the navigation (copy any `<a href="...">` link)
